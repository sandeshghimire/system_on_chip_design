Installation Instructions for the SoC Workshop
----------------------------------------------

1. Create a folder c:\altera_trn. Ensure the folder name is an exact match as described.

2. Download the SoCKIT_Materials.zip file and place in 
c:\altera_trn

3. Unzip SoCKIT_Materials.zip to the local folder.

4. Implement the instructions in Module 1 of the SoC_HW_Lab_13.0.pdf guide.

5. Implement the instructions in Module 1 of the SoC_SW_Lab_13.0.pdf guide.
